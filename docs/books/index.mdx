# 书库

收集整理的免费书籍资源，方便查找和阅读。

:::warning
本站仅提供链接，不存储书籍内容。所有资源均来自公共领域或开源项目。
:::

## 分类

- [技术书籍](./tech/) - 编程、计算机科学相关
- [经典文学](./classics/) - 公共领域的文学作品
- [哲学思想](./philosophy/) - 哲学、思想类书籍

## 主要资源站

- [Project Gutenberg](https://www.gutenberg.org/) - 7万+免费电子书
- [维基文库](https://zh.wikisource.org/) - 中文文献
- [GitHub Free Programming Books](https://github.com/EbookFoundation/free-programming-books) - 编程书籍列表
- [Internet Archive](https://archive.org/) - 数字图书馆

## 说明

所有书籍均为公共领域作品或采用开源许可证，可以自由阅读。点击书名跳转到原始站点。
