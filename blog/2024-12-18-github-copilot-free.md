GitHub 宣布 Copilot 对个人用户免费了。这是个大新闻。

<!-- truncate -->

## 免费计划

从 2024 年 12 月开始，GitHub Copilot 推出免费版：

- **每月 2000 次代码补全**
- **每月 50 次聊天**
- 支持 VS Code、Visual Studio、Neovim 等编辑器

付费版（$10/月）不限次数。

## 为什么免费

可能原因：

**1. 竞争压力**

- Cursor 很火，月费 $20 但体验更好
- 国内的通义灵码、文心快码都免费
- 不免费就没人用了

**2. 培养用户习惯**

免费让更多人用上，习惯了就离不开。以后再收费或者推增值服务。

**3. 数据收集**

更多用户意味着更多数据，能改进模型。

## 实际体验

我用了几天，感觉：

**优点**：
- 代码补全挺准
- 支持多种语言
- 和 GitHub 集成好

**缺点**：
- 2000 次补全不够用（一天就用完了）
- 聊天功能比 Cursor 差
- 有时候建议很蠢

## 对比 Cursor

Cursor 虽然贵（$20/月），但：

- 不限次数
- 聊天功能强
- 能理解整个项目

如果预算够，Cursor 更值。

## 对开发者的影响

AI 编程工具已经是标配了：

- 提高效率 30-50%
- 减少重复劳动
- 但还是要懂原理

不会用 AI 工具的程序员，竞争力会下降。

## 总结

GitHub Copilot 免费是好事，但：

- 免费版限制多
- 体验不如 Cursor
- 适合尝鲜，不适合重度使用

想认真用 AI 编程，还是得付费。
