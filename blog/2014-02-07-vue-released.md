尤雨溪发布了 Vue.js，一个轻量级的前端框架。比 React 和 Angular 都简单。

<!-- truncate -->

## 什么是 Vue

Vue 是个渐进式框架：

- 可以只用一部分功能
- 学习曲线平缓
- 中文文档友好

## 核心特点

**1. 响应式数据**

数据变了，视图自动更新：

```js
new Vue({
  data: {
    message: 'Hello'
  }
})
```

**2. 组件化**

和 React 一样，支持组件化开发。

**3. 模板语法**

用 HTML 模板，不用学 JSX：

```html
<div>{{ message }}</div>
```

## 对比 React 和 Angular

| 特性 | Vue | React | Angular |
|------|-----|-------|---------|
| 学习曲线 | 平缓 | 中等 | 陡峭 |
| 灵活性 | 高 | 高 | 低 |
| 生态 | 中 | 强 | 强 |

Vue 最大的优势是简单。

## 为什么火

**1. 简单易学**

新手友好，上手快。

**2. 中文文档**

尤雨溪是中国人，中文文档很好。

**3. 渐进式**

可以逐步引入，不用一次性重写。

## 在中国特别火

Vue 在中国的流行度超过 React：

- 中文文档好
- 社区活跃
- 大厂在用（阿里、腾讯）

## 争议

**1. 个人项目**

早期是尤雨溪一个人维护，有人担心可持续性。

**2. 生态不如 React**

第三方库没 React 多。

## 后续发展

**Vue 2.0**

2016 年发布，性能大幅提升。

**Vue 3.0**

2020 年发布，用 TypeScript 重写，性能更好。

## 总结

Vue 是个成功的开源项目：

- 简单易学
- 渐进式设计
- 在中国特别火

证明了个人也能做出优秀的框架。
