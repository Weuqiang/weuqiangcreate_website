### Error

`Error`

- `message`
- `name`
- `stack`

``` js
class MyError extends Error {
    constructor(...props) {
        super(...props)
        this.name = 'MyError'
    }
}

try {
    throw new MyError('myMessage')
} catch (error) {
    console.error(error.name + ': ' + error.message)
    console.error(error.stack)
}
```
