选择器​

/* 基础选择器 */
div { }                    /* 标签选择器 */
#root { }                  /* ID 选择器 */
.container { }             /* class 选择器 */
* { }                      /* 通配符选择器 */

/* 组合选择器 */
div p { }                  /* 后代选择器 */
div > p { }                /* 子代选择器 */
div + p { }                /* 相邻兄弟选择器 */
div ~ p { }                /* 通用兄弟选择器 */

/* 属性选择器 */
input[type="text"] { }
a[href^="https"] { }       /* 以 https 开头 */
a[href$=".pdf"] { }        /* 以 .pdf 结尾 */
a[href*="example"] { }     /* 包含 example */

/* 伪类选择器 */
a:hover { }
li:first-child { }
li:last-child { }
li:nth-child(2n) { }       /* 偶数项 */
li:nth-child(odd) { }      /* 奇数项 */
input:focus { }
input:disabled { }

/* 伪元素选择器 */
p::before { }
p::after { }
p::first-line { }
p::first-letter { }
::selection { }

选择器优先级​

!important > 内联样式 > ID 选择器 > class 选择器 > 标签选择器 > 通配符 > 继承

计算规则：

  • 内联样式：1000
  • ID 选择器：100
  • class/属性/伪类：10
  • 标签/伪元素：1
/* 优先级：1 */
div { }

/* 优先级：10 */
.container { }

/* 优先级：100 */
#root { }

/* 优先级：11 */
div.container { }

/* 优先级：111 */
#root div.container { }

盒模型​

每个元素由 content、padding、border、margin 组成。

.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

box-sizing​

/* 标准盒模型（默认） */
.content-box {
  box-sizing: content-box;
  width: 200px;  /* 实际宽度 = 200 + padding + border */
}

/* IE 盒模型 */
.border-box {
  box-sizing: border-box;
  width: 200px;  /* 实际宽度 = 200（包含 padding 和 border） */
}

推荐做法：全局使用 border-box

* {
  box-sizing: border-box;
}

定位​

/* 静态定位（默认） */
.static {
  position: static;
}

/* 相对定位：相对于自身原始位置 */
.relative {
  position: relative;
  top: 10px;
  left: 20px;
}

/* 绝对定位：相对于最近的非 static 祖先元素 */
.absolute {
  position: absolute;
  top: 0;
  right: 0;
}

/* 固定定位：相对于视口 */
.fixed {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

/* 粘性定位：滚动到阈值时固定 */
.sticky {
  position: sticky;
  top: 0;
}

布局​

Flexbox​

容器属性：

.container {
  display: flex;
  
  /* 主轴方向 */
  flex-direction: row;           /* row | row-reverse | column | column-reverse */
  
  /* 换行 */
  flex-wrap: nowrap;             /* nowrap | wrap | wrap-reverse */
  
  /* 主轴对齐 */
  justify-content: flex-start;   /* flex-start | flex-end | center | space-between | space-around | space-evenly */
  
  /* 交叉轴对齐 */
  align-items: stretch;          /* stretch | flex-start | flex-end | center | baseline */
  
  /* 多行对齐 */
  align-content: flex-start;     /* flex-start | flex-end | center | space-between | space-around | stretch */
  
  /* 简写 */
  flex-flow: row wrap;           /* flex-direction flex-wrap */
}

子元素属性：

.item {
  /* 排序 */
  order: 0;                      /* 数字越小越靠前 */
  
  /* 放大比例 */
  flex-grow: 0;                  /* 默认 0，不放大 */
  
  /* 缩小比例 */
  flex-shrink: 1;                /* 默认 1，自动缩小 */
  
  /* 初始大小 */
  flex-basis: auto;              /* auto | 200px | 50% */
  
  /* 简写 */
  flex: 1;                       /* flex-grow flex-shrink flex-basis */
  flex: 1 1 auto;
  
  /* 单独设置交叉轴对齐 */
  align-self: auto;              /* auto | flex-start | flex-end | center | baseline | stretch */
}

常见布局：

/* 水平垂直居中 */
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 两端对齐 */
.space-between {
  display: flex;
  justify-content: space-between;
}

/* 等分布局 */
.equal {
  display: flex;
}
.equal > * {
  flex: 1;
}

/* 自动 margin */
.auto-margin {
  display: flex;
}
.auto-margin .item:last-child {
  margin-left: auto;  /* 推到最右边 */
}

Grid​

Grid 是二维布局系统，适合复杂布局。

.container {
  display: grid;
  
  /* 定义行和列 */
  grid-template-columns: 200px 1fr 1fr;     /* 3列 */
  grid-template-rows: 100px auto 100px;     /* 3行 */
  
  /* 简写 */
  grid-template-columns: repeat(3, 1fr);    /* 3个等宽列 */
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));  /* 自适应 */
  
  /* 间距 */
  gap: 20px;                                /* 行列间距 */
  row-gap: 20px;                            /* 行间距 */
  column-gap: 10px;                         /* 列间距 */
  
  /* 对齐 */
  justify-items: start;                     /* 水平对齐 */
  align-items: start;                       /* 垂直对齐 */
  justify-content: center;                  /* 整体水平对齐 */
  align-content: center;                    /* 整体垂直对齐 */
}

.item {
  /* 指定位置 */
  grid-column: 1 / 3;                       /* 占据第1-2列 */
  grid-row: 1 / 2;                          /* 占据第1行 */
  
  /* 简写 */
  grid-area: 1 / 1 / 2 / 3;                 /* row-start / col-start / row-end / col-end */
  
  /* 对齐 */
  justify-self: center;
  align-self: center;
}

常见布局：

/* 圣杯布局 */
.holy-grail {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

/* 卡片网格 */
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 20px;
}

/* 12列栅格系统 */
.grid-12 {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 20px;
}
.col-6 {
  grid-column: span 6;
}

动画​

Transition​

过渡动画，适合简单的状态变化。

.box {
  width: 100px;
  height: 100px;
  background: blue;
  
  /* 单个属性 */
  transition: width 0.3s ease-in-out;
  
  /* 多个属性 */
  transition: width 0.3s, height 0.3s, background 0.5s;
  
  /* 所有属性 */
  transition: all 0.3s ease;
  
  /* 完整写法 */
  transition-property: width;
  transition-duration: 0.3s;
  transition-timing-function: ease-in-out;
  transition-delay: 0.1s;
}

.box:hover {
  width: 200px;
  height: 200px;
  background: red;
}

timing-function 值：

  • linear：匀速
  • ease：慢-快-慢（默认）
  • ease-in：慢-快
  • ease-out：快-慢
  • ease-in-out：慢-快-慢
  • cubic-bezier(0.1, 0.7, 1.0, 0.1)：自定义贝塞尔曲线

Animation​

关键帧动画，适合复杂动画。

@keyframes slide {
  0% {
    transform: translateX(0);
    opacity: 0;
  }
  50% {
    opacity: 1;
  }
  100% {
    transform: translateX(100px);
    opacity: 0;
  }
}

.box {
  animation: slide 2s ease-in-out infinite;
  
  /* 完整写法 */
  animation-name: slide;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-delay: 0s;
  animation-iteration-count: infinite;  /* 或具体次数 */
  animation-direction: normal;          /* normal | reverse | alternate | alternate-reverse */
  animation-fill-mode: forwards;        /* none | forwards | backwards | both */
  animation-play-state: running;        /* running | paused */
}

常见动画：

/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 弹跳 */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

/* 旋转 */
@keyframes rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

/* 脉冲 */
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

  }

响应式单位​

.responsive {
  /* 相对单位 */
  width: 50%;              /* 相对父元素 */
  font-size: 1.5em;        /* 相对父元素字体大小 */
  font-size: 1.5rem;       /* 相对根元素字体大小 */
  
  /* 视口单位 */
  width: 50vw;             /* 视口宽度的 50% */
  height: 100vh;           /* 视口高度的 100% */

  font-size: 5vmax;        /* vw 和 vh 中较大的值 */

}

BFC（块级格式化上下文）​

触发 BFC 的方式：

  • overflow: hidden/auto/scroll
  • display: inline-block/flex/grid
  • position: absolute/fixed
  • float: left/right

BFC 的作用：

  • 清除浮动
  • 防止 margin 重叠
  • 自适应两栏布局
/* 防止 margin 重叠 */
.container {
  overflow: hidden;  /* 创建 BFC */
}

/* 自适应两栏布局 */
.left {
  float: left;
  width: 200px;
}
.right {
  overflow: hidden;  /* 创建 BFC，不被浮动元素覆盖 */
}

性能优化​

减少重排和重绘​

/* 使用 transform 代替 top/left */
.move {
  transform: translateX(100px);  /* 好 */
  /* left: 100px; */              /* 差 */
}

/* 使用 opacity 代替 visibility */
.fade {
  opacity: 0;                     /* 好 */
  /* visibility: hidden; */       /* 差 */
}

/* 使用 will-change 提示浏览器 */
.animate {
  will-change: transform, opacity;
}

常见问题​

display: none vs visibility: hidden vs opacity: 0​

属性占据空间响应事件影响子元素性能
display: none否否是回流+重绘
visibility: hidden是否可覆盖重绘
opacity: 0是是是GPU 加速
